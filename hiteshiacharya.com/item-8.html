3D shape descriptor, shape segmentation, registration and joint matching

September 2015

Achieved efficient segmentation of 3D models using k –center, k –mean, and center shift algorithms.

Implemented bag of features (BOF) model data-driven shape descriptor, named bag-of-feature descriptor (BOFD) for 3D shape retrieval using a shape search engine.

Developed a 3D shape search engine for a database of 588 3D models. It involved the comparison of Euclidean and geodesic shape descriptors of the query with the model dataset to retrieve the top 5 matches.

Used Hungarian algorithm to register point correspondence between human 3D models with different body movement and then implementing cyclic joint matching.

Tools: Matlab, Matlab GUI
