Retrieving 3D objects from indoor scene

November 2015

The project involves detecting 2D objects in an indoor scene, then retrieving the corresponding 3D object from a database using 3D rendering and feature matching. This has a scope in 2D to 3D scene transformation and RGBD image processing.

Used GALIF, SIHKS features and Cross domain manifolds. Also used 3D rendering to obtain 360 deg views of 3D models and store them as 2D images for further processing.

Tools: C++, OpenCV and MATLAB
